# 型别推导
```c++
template<typename T>
void f(ParamType param);
```
---
### 顶层和底层const（前置知识）
底层const
>* 限制内容（指向的数据）
>* 不能修改指向的值
```c++
const char* p = "hello";
*p = 'a';  // ❌ 错误：不能修改内容
p = "world";  // ✅ 可以：可以改变指向
```

顶层const
>* 限制指向（指针本身）
>* 不能修改指针的指向
```c++
char* const p = "hello";
p = "world";  // ❌ 错误：不能改变指向
*p = 'a';     // ✅ 可以：可以修改内容
```
---
### ParamType既非指针也非引用
```c++
template<typename T>
void f(T param); 	// param 按值传递

const char* const ptr = "Func With Ptr";

f(ptr);				// T：const char*
					// 传入的ptr经过拷贝，常量属性被忽略
```
案例：
```c++
template <typename T>
void f(T param) {
    param = "aaa";  // 正确，修改的是局部副本
	*param = "bbb"; // 错误，无法修改内容
}

int main() {
    const char* const ptr = "avc vvv";
    f(ptr);
    
    std::cout << ptr << std::endl;  // 输出: avc vvv (没变！)
									// 因为传入f的是ptr的拷贝
}
```
